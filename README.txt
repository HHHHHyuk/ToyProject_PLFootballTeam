## 프리미어 리그 축구팀관리 시스템 ##
1. 개요
 - 해외축구의 팬으로서 관심있는 분야를 선정하여 프로젝트의 흥미 유발.
 - 강의로 배운 JPA, SPRING DATA JPA, QUERYDSL을 활용하여 Restful로 동작하는 프로그램 구현.
 - 시즌별 PL팀에 대한 데이터와 선수들의 데이터를 다양한 조회방식을 통해서 서비스 제공.

 

2. 목적
 - 강의로 배운 Spring boot와 JPA를 활용하여 복습하고, 자연스럽게 활용능력을 증대.
 - Github에 프로젝트 추가

 

3. 프로그램 주요 기능
 - 팀 : 팀 등록,수정,삭제 // 팀별 순위정보( 경기수, 승점, 승, 무, 패, 득점, 실점, 득실차 )
 - 선수 : 선수 등록,수정,삭제 // 선수별 순위정보( 득점, 도움, 공격포인트, 슈팅, 파울, 경고, 퇴장, 코너킥, 패널티킥, 오프사이드, 유효슈팅, 경기수 ) 

 

4. Role
 - 선수는 반드시 팀을 가지고 있어야한다. ( 등록시, 팀 선택 )
 - 한 팀에서 중복된 등번호는 불가능하다. 
 - 시즌별 팀은 최대 20팀이 등록이 가능하다.
